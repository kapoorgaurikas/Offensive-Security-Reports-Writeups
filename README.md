# Offensive-Security-Reports-Writeups


Welcome to my penetration testing report repository.

This repository contains penetration testing reports created while solving vulnerable machines from various cybersecurity training platforms.

## Platforms

- TryHackMe
- Offsec
- HackTheBox
- PortSwigger

## Report Structure

Each report includes:

- Executive Summary
- Scope & Methodology
- Reconnaissance
- Enumeration
- Initial Access
- Post Exploitation
- Privilege Escalation
- Findings Summary
- Remediation Recommendations

## Disclaimer

All reports were created in authorized laboratory environments for educational purposes only.

Challenge flags and sensitive information have been redacted.
